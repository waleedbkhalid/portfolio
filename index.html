Research & Publications

 Legged Robot Navigation in uncertain terrains using Deep Reinforcement Learning

Worked on this legged robot navigation project with Dr. Sehoon Ha at Georgia Tech. Developed and vectorized the robot environment in the RaiSim simulator using C++ and used Python bindings for integration with the deep RL training code written in Pytorch. Achieved over 95% success rate for the baseline point-to-goal navigation policy on a flat plane using PPO. Created different terrains with varying features to make a robot traversal dataset in uncertain terrains for training an imitation learning policy, which is the future works for the project

Course Projects

 Cartpole Control Using Deep Reinforcement Learning.
Georgia Institute of Technology | CS 7643-Deep Learning course project

Did a comparative study of Deep Q-learning and it's variations and PPO on the classical cartpole control problem using Open AI gym.

Manipulator Control using Deep Reinforcement and Imitation Learning.
Georgia Institute of Technology | CS 8803 - Deep Reinforcement Learning course project

Studied manipulator control using reinforcement and imitation learning via MLPs, CNNs, and transformers on the ManiSkill2 dataset. Reduced the covariate shift of the performance using sequential learning and obtained a 20 percent success rate as well. Implemented PPO-based RL control and achieved 100 percent accuracy for a selected task.

Legged Robot and Humanoid Robot Control using Deep Reinforcement Learning and Imitation Learning in MuJoCo.
Georgia Institute of Technology | CS 8803 - Deep Reinforcement Learning Assignments

Implemented legged robot control with RL, curriculum learning, and custom reward functions to make a quadruped robot move over obstacles with increasing height. Implemented two walking modes (normal and stealthy) for a humanoid robot using imitation learning and reference state intialization.

Simultaneous Location and Mapping using Turtlebot 3 and ROS 2
Georgia Institute of Technology | ECE 7785-Introduction to Robotics Research course project

Developed a map by teleoperating the robot in the physical and Gazebo environment. Tuned navigation stack parameters in ROS2 for the robot to traverse the map smoothly and reach 3 randomly provided locations on the map.
Robot Maze Navigation for variable maze strcutures
Georgia Institute of Technology | ECE 7785-Introduction to Robotics Research course project

The main challenge was that the maze used for the demonstration was unknown. Implemented a combination of Bug algorithms, collision avoidance, and CNN-based Image classification to make a solution that was invariant to the maze structure, starting and ending point.

Road Image Segmentation with Supervised & Unsupervised methods.
Georgia Institute of Technology | CS 7641-Machine Learning course project

Deployed different supervised & unsupervised methods for segmentation of road images in autonomous driving. The methods employed included K-Means, DB scan, Gaussian Mixture Models, and U-NET. The main libraries used included Scikit-Learn, Numpy, and PyTorch. The dataset used was the CARLA road images dataset
Pedestrian Behavior Classification.
Habib University | Computer Vision course project

Created the image dataset from multiple publicly available pedestrian image datasets. Deployed Faster RCNN to detect pedestrians. Used transfer learning with VGG16 and Efficient Net to classify pedestrians as distracted or Non-distracted. The main libraries used included OpenCV & tensorflow.

Autonomous Line Following Ball Throwing Robot.
Habib University | Microcontrollers and Interfacing course project

Developed this robot with Arduino, IR sensors, self-fabricated motor drivers, and regulators using discrete electronic components. Used a differential drive for mobility, and a flywheel mechanism for shooting.
HAWAI FIRING: Shooting game developed using C++
Habib University | Object Oriented Programming course project

Developed this game using C++ and SDL 2.0 library. The primary goal was to exercise the OOP design principles, maintain clean code practices, design and implement UML diagrams for code maintenance, and develop game graphics from scratch.
