Legged Robot Navigation in Uncertain Terrains using Deep Reinforcement Learning.
Georgia Institute of Technology | Research Project

Worked on this legged robot navigation project with Dr. Sehoon Ha at Georgia Tech. Developed and vectorized the robot environment in the RaiSim simulator using C++ and used Python bindings for integration with the deep RL training code written in PyTorch. Achieved over 95% success rate for the baseline point-to-goal navigation policy on a flat plane using PPO. Created different terrains with varying features to make a robot traversal dataset in uncertain terrains for training an imitation learning policy, which is the future works for the project.
